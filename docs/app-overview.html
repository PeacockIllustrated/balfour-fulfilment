Customisation Features

Beyond the standard catalogue, the portal supports fully bespoke signage and non-standard dimensions.

//
Custom Sign Builder
Design bespoke signs from scratch. Choose sign type (warning, prohibition, mandatory, information, fire safety, directional, security, environmental), shape, material, and enter custom text. A live preview renders in real time. Submitted as a quote request.
<>
Custom Sizing
Enter non-standard dimensions on any product. The system automatically matches to the nearest standard variant for instant pricing, or flags for a manual quote if no close match exists. Area tolerance matching within 150%.
Aa
Custom Fields
Products that require site-specific data (site name, phone number, address, etc.) present dynamic input fields at the point of ordering. Values are captured in the order and carried through to fulfilment.
£
Pricing Logic
Standard items priced from catalogue. Custom signs submitted at £0 pending quote. Custom sizes auto-matched to nearest variant price. VAT (20%) calculated server-side. Prices validated server-side to prevent tampering.

Integrations & Security

The portal connects to external systems for email delivery, fulfilment routing, and automated workflows.

Email & Automation

All transactional emails are triggered via Make.com webhooks, providing a flexible automation layer between the portal and downstream systems.

  • Order confirmation — sent to contact on submission
  • Internal PO email — full order details to production team
  • Purchaser notification — PO approval request with upload link
  • All emails are branded HTML with item tables, pricing, and images
  • Custom sign items colour-coded by type in emails

Fulfilment Integration

"Send to Nest" action fires a webhook to route the order into the production system with complete order data, item details, and PO references.

Security

  • Password-based authentication (shop & admin tiers)
  • HTTP-only secure cookies with 30-day expiry
  • Rate-limited login (5 attempts per minute per IP)
  • Server-side price recalculation (prevents client tampering)
  • Token-protected PO upload links (time-bound)
  • Supabase Row Level Security on all tables
  • Security headers: X-Frame-Options, HSTS, CSP

Hosting & Deployment

  • Deployed on Vercel with automatic Git deployments
  • Environment variables for all secrets and config
  • PostgreSQL database hosted on Supabase
  • Global CDN for static assets and product images
  • Zero-downtime deployments on every push

The Make.com integration layer means email templates, routing rules, and downstream system connections can be modified without code changes to the portal itself.
